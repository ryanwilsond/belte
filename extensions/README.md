# Extensions

`./belte` is a work in progress VSCode extension that provides an LSP for Belte, eventually planning to include
diagnostic display, basic intellisense, renaming, and reference tracking.

`./belte-noslp` is a simple VSCode extension that provides syntax highlighting that can be used while the LSP version
is under development. Because this extension is not published, it can be used by copying the folder into your VSCode
extensions directory.
