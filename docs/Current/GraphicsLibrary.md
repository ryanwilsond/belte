# 9 Graphics Library

The Graphics Library is a very simple 2d graphics library. The graphics library makes use of the `Update` method to
run code every frame. See also [Program and Update](ControlFlow.md#222-program-and-update).

The Graphics API can only be invoked when compiling with the
[`--type=graphics` option](../Buckle.md#--typeconsolegraphics-default-console).
